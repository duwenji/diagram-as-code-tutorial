# コンポジション
テキストコンポジション:
- タイトルとサブタイトルは上部ゾーンに配置し、イラストエリアと明確に分離する。
- タイトルゾーンはフローチャートの開始ノード（角丸矩形）を想起させるデザインと連動する。
- テキストは清潔感のある左寄せまたは中央揃えで、図の階層構造を体現する。
- Mermaid基礎からプロジェクト開発フェーズまでの学習パスを示すビジュアル構造がタイトルの補強となる。

ビジュアルコンポジション:
- メインビジュアル: 角丸四角形のノードと細い曲線エッジで構成されたフローチャートとネットワークグラフが重なり合う構図。各ノードは異なる色で塗り分けられ、Mermaid/Graphvizによる設計表現の層構造を視覚化する。
- 中央下部: 左から右へ流れる開発フェーズのタイムライン（要件定義→設計→実装→リリース）を重ねたグラフ構造。プロジェクト開発フェーズ別図解のカテゴリを象徴する。
- 背景: 淡いグリッドパターン（点線または実線）でDOT言語のレイアウト制御の概念を背景テクスチャとして取り入れる。
- アクセント: コードエディタ風の角丸矩形フレームで「Diagrams as Code」というコードで図を描くコンテキストを示す。
- レイアウト: 上部20%をテキスト、下部80%をフローチャート・ネットワークグラフ・開発フェーズタイムラインの複合イラストに割り当て。
- デコレーション: 図の構文キーワード（flowchart / digraph / -->）を極小テキストとして各要素近傍に配置し、チュートリアルの技術的性質を強調する。
